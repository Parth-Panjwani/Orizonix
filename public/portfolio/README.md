# Portfolio Images

Place your project images in this directory.

## Structure

For each project, add images to the `public/portfolio/` directory following this naming convention:

- `techstartup-1.jpg`, `techstartup-2.jpg`, etc.
- `ecommerce-1.jpg`, `ecommerce-2.jpg`, etc.
- `automation-1.jpg`, `automation-2.jpg`, etc.
- `social-1.jpg`, `social-2.jpg`, etc.
- `saas-1.jpg`, `saas-2.jpg`, etc.
- `telegram-bot-1.jpg`, `telegram-bot-2.jpg`, etc.

## Image Requirements

- Recommended format: JPG or PNG
- Recommended size: 1200x800px or higher
- Aspect ratio: 3:2 or 16:9
- File size: Keep under 500KB per image for optimal performance

## Updating Projects

Edit `/data/portfolio.ts` to update project information and image paths.
